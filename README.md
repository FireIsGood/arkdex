# [Arkdex](https://fireisgood.github.io/arkdex/)

Arknights operator viewer made with [Astro](https://astro.build) made by FireIsGood#0733

## To run

```bash
npm run dev
```

Runs on port 3000

## To build

```bash
npm run build
```

Creates files to `/docs` folder for github pages. NOTE: DEPLOYS TO /arkdex
